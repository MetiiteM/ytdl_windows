# YTDL Windows

Simple YouTube downloader for Windows powered by **yt-dlp** and **FFmpeg**.

## Features

- Download YouTube videos
- Download audio as MP3
- Multiple video quality options
- Fast and lightweight
- Automatic metadata support
- Easy Windows setup

## Requirements

- Python 3.10+
- FFmpeg

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

Using the included batch file:

```bat
run_windows.bat
```

Or manually:

```bash
python ytdl_windows.py
```

## Build EXE

```bat
build_windows_exe.bat
```

## Supported Formats

### Video
- MP4

### Audio
- MP3

## Notes

- Some videos may require cookies for downloading.
- Make sure FFmpeg is installed and available in your system PATH.
- Keeping yt-dlp updated is recommended for the best compatibility.
